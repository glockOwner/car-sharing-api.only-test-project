Тестовое задание для компании only, вакансия Php-разработчик(Laravel). 

Перед запуском проекта необходимо установить: 
* php8.1
* Composer
* docker

  API метод получения списка доступных текущему пользователю на запланированное время автомобилей доступен по пути **localhost:88/api/available-cars**

  **Для отправления запроса используется аутентификация с bearer токеном. Поэтому необходимо создать токен для пользователя, от лица которого будете выполнять запрос. Это делается с помощью команды ```php artisan app:create-token {id}```**
